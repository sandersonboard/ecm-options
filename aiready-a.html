What's happening underneath. Every event lands in the warehouse, the classifier tags it against the control catalog, and the tagged event is filed against any control whose evidence requirements it satisfies — before the quarter closes, not after. The owner doesn't go find evidence; the evidence finds the control.

State 2. Events stream in continuously. The r.taylor escalation that became a finding in ECM options's evidence flow? It would have been tagged against control 4.2.1 the second it happened — and ECM would have flagged the missing approval ticket the same moment, not two quarters later in review.
